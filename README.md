# Adaptive Sampling for Edge Computing

## Overview
This project proposes adaptive sampling techniques to reduce computational cost in edge computing systems.

## Methods
- Random Sampling
- Stratified Sampling
- Systematic Sampling
- Cluster Sampling
- Moving Average & Gaussian Smoothing

## Results
- Sampling significantly reduces training time
- Stratified sampling provides best accuracy-efficiency trade-off
- Hybrid approach improves performance further
